# SmartZone Battle Version

Готовий бойовий статичний магазин для GitHub Pages.

## Що є
- Головна сторінка
- Каталог товарів
- Категорії
- Пошук і сортування
- Кошик
- Оформлення замовлення
- Адмін-панель
- Додавання / редагування / видалення товарів
- Перегляд замовлень

## Вхід в адмінку
- URL: `/admin.html`
- Пароль: `SmartZone2026`

## Важливо
Це статичний сайт для GitHub Pages, тому дані зберігаються у LocalStorage браузера.
