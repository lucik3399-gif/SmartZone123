# SmartZone Premium

Преміальна статична версія магазину SmartZone для GitHub Pages.

## Файли
- `index.html` — головна сторінка магазину
- `admin.html` — адмін панель
- `app.js` — логіка магазину
- `admin.js` — логіка адмінки
- `styles.css` — стилі сайту

## Адмінка
- сторінка: `admin.html`
- пароль: `SmartZone2026`

## Важливо
Це статичний сайт для GitHub Pages. Дані товарів, кошика та замовлень зберігаються в браузері через localStorage.
